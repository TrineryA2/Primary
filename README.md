# Primary

**A color memory game that tests your eye for detail.**

Primary is a minimalist color memory game. Memorize a target color, then recreate it using RGB sliders. The closer your match, the higher your score.

---

## Features

- **Memorize and recreate** – Each round challenges your color memory.
- **RGB sliders** – Fine-tune your match with precision.
- **Accuracy scoring** – See exactly how close your guess was.
- **Best score saved** – Your highest score stays on your device.

---

## Platform

- **Android APK** – Packaged as a lightweight Web-to-App APK.

---

## Built With

- HTML5, CSS3, Vanilla JavaScript
- Local storage for offline data

---

## How to Install

1. Download the latest `.apk` from the Releases page.
2. Install it on your Android device (allow unknown sources).
3. Play.
